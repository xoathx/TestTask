# Тестовое задание — Скандинавия

Проект с автотестами для компании **Скандинавия**.

---

## Требования

Для работы необходимы:
- **Maven**  
- **JDK 21**  
- **JRE 21**

---

## Запуск тестов

```bash
mvn clean test
mvn allure:report
mvn allure:serve
```

## Стек технологий

- **Java 21**
- **JUnit 5**
- **RestAssured**
- **Allure**
